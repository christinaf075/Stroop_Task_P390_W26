# Stroop_Task_P390_W26
A Stroop Task Study coded on Pygame for a PSYCH seminar course

## Final Products
- [Presentation](https://christinaf075.github.io/Stroop_Task_P390_W26/stroop_presentation/stroop_presentation.html)
- [Manuscript](https://christinaf075.github.io/Stroop_Task_P390_W26/stroop_journalarticle/stroop_article.pdf)

## Prereqs
Ensure you have the following installed on your device:
- Python (3.10+ recommended)
- R (any version after 4.0)

## Clone the repo

```bash
git clone https://github.com/h364lee/psych420_q_learning.git](https://github.com/christinaf075/Stroop_Task_P390_W26.git
```

## Setup
1. Go to the main directory of the repo in your terminal
2. Create a virtual environment by entering: python3 -m venv stroop
3. Activate it: source stroop/bin/activate
4. Install required packages by entering: pip install -r requirements.txt

## Navigation
1. Open `stroop_experiment.py` to run the experiment.
2. Go to Data to view collected data.
3. Go to stroop_presentation to view presentation files.
4. Go to stroop_journalarticle to view manuscript files.
